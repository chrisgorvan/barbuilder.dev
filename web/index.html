Why BarBuilder?

Everything you need to ship beautiful progress indicators, instantly.

Infinite Customisation

Customise colours, styles, shapes and labels to match your brand. Choose from classic bars, pill, minimal, badge, segments, icon ratings and more.

Easy API & Embedding

Just a URL. Embed in GitHub READMEs, documentation, dashboards, emails and anywhere that renders images — no JavaScript needed.

Responsive Design

Bars scale to any width with crisp SVG rendering. They look sharp on retina screens, in dark mode, and across all devices and browsers.

Live Real-Time Updates

Update your bar by changing the URL value. The image updates immediately everywhere it's embedded — no cache-busting or redeployment required.

Showcase

A few examples of what you can build with BarBuilder.

GitHub Build Status

Badge-style bars are ideal for GitHub READMEs. Show build status, test coverage, or release progress at a glance.

Star Rating

Icon-type bars support half steps. Perfect for ratings, scores, and satisfaction metrics with a visual star or custom shape.

Task Tracker

Segmented X-of-Y bars let you track discrete progress — tasks completed, milestones reached, or steps in a workflow.

Download Counter

Show X-of-Y progress for download goals, funding targets, or any metric with a defined total.

Test Coverage

Pill-style bars give a softer, modern look. Great for test coverage badges, health scores, and completion rates.

Dark Theme

Every bar supports a dark theme — ideal for dark-mode READMEs, dashboards, and documentation sites.
